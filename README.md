🎓 Edufonica Solutions — Website
A modern, responsive single-page website for Edufonica Solutions — an education growth consultancy helping schools and colleges start, scale, and digitize across India.

🔗 Live Site: edufonica-website.vercel.app

✨ Features
Fully Responsive — Works seamlessly on desktop, tablet, and mobile devices
Single-Page Design — Smooth scroll navigation between Home, About, Services, and Contact sections
Animated Reveal Effects — Sections fade in as you scroll down the page
Service Search Filter — Real-time search bar to filter across 12 service offerings
Glassmorphism UI — Floating glass-panel stat cards on the hero section
WhatsApp Integration — Contact form redirects enquiries directly to WhatsApp
Mobile Hamburger Menu — Collapsible navigation for small screens
Google Fonts — Poppins (headings) + Inter (body) for clean, professional typography
🛠️ Tech Stack
Technology	Purpose
HTML5	Page structure and semantic markup
CSS3	Styling, animations, glassmorphism effects, responsive grid
JavaScript	Scroll animations, mobile menu toggle, service search filter
Google Fonts	Poppins & Inter typefaces
Vercel	Deployment and hosting

🚀 How to Run Locally
No build tools or frameworks needed — it's plain HTML/CSS/JS.

Clone the repository

bash

git clone https://github.com/Harshvt8585/edufonica-website.git
Open in browser

bash

cd edufonica-website
Then simply open index.html in your browser. You can also use a local server:

bash

# Using Python
python3 -m http.server 8000
# Then visit http://localhost:8000
